# **App Name**: WebStart

## Core Features:

- PWA Configuration: Configures the Next.js application as a Progressive Web App (PWA) with a manifest.json file.
- Manifest Generation: Generates a manifest.json file that includes essential PWA configurations such as `start_url` set to `/` and `display` set to `standalone`.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to give a clean and calming impression.
- Background color: Light gray (#F0F4F8) for a neutral and modern backdrop.
- Accent color: Light green (#A0EBA0) for interactive elements, and a touch of freshness.
- Font: 'Inter', a sans-serif font providing a modern and neutral look, suitable for both headlines and body text.
- Clean and minimal layout with a focus on content. Ample white space to ensure readability and reduce clutter.
- Simple, outlined icons to match the modern aesthetic.
- Subtle, smooth transitions for a polished user experience.
